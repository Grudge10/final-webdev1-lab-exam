# My Dynamic Portfolio

A responsive, JSON-driven personal portfolio made by **Jan Liam E. Arias** built with vanilla HTML, CSS, and JavaScript.

## Structure

```
index.html        # Page structure
data/data.json    # All content lives here
css/styles.css    # Styling (mobile-first, dark brown theme)
js/main.js        # Fetches JSON, renders content, interactions
assets/           # Profile photo + project thumbnails
```

## Run it yourself

Use a netlify, github pages, vercel, etc.

## Customize

Edit `data/data.json` - adding a new project, skill, or timeline entry automatically renders it on the page. No code changes needed.
